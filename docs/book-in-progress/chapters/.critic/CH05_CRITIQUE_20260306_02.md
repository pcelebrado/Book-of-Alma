## Findings

### CRITICAL: Vanna sign convention + flow table contradict Alma and can miseducate
- **Location**: OP3NF1XER/nate-alma/book-in-progress/chapters/CH05.md:85-116
- **Issue**: The chapter defines positive/negative vanna in terms of "long OTM calls/short OTM puts" vs "long OTM puts/short OTM calls" and then provides a 2×2 table stating how dealer delta changes with IV rising/falling. This mapping conflicts with Alma's explicit statement in the Sept 18 post that "If vol is sold, negative vanna would make the dealer longer delta, making him sell futures, and vice versa." The table currently implies the opposite behavior in the negative-vanna / IV-falling quadrant.
- **Evidence**:
  - CH05 table: "Dealer Net Vanna = Negative" and "IV Falling" => "Dealers get longer delta → Buy futures into weakness (supportive)".
  - Bible: "If vol is sold, negative vanna would make the dealer longer delta, making him sell futures" (selling futures, not buying).
- **Required Fix**: Rewrite Section 5.3 so that:
  1. Any vanna sign convention is explicitly stated and does not rely on incorrect option-type heuristics.
  2. The flow mapping is either removed OR rebuilt directly from the Sept 18 Bible passage with matching language and conditioning ("in this schematic example...").
  3. The chapter clearly distinguishes: delta change direction vs hedge trade direction (buy/sell futures) to avoid sign confusion.
